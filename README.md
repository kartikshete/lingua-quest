# 🌍 LinguaQuest
An engaging language-learning web application built to make mastering new vocabularies fully gamified and interactive.

## 📝 Overview
LinguaQuest turns language acquisition into an adventure. Equipped with visually impressive celebration animations (like confetti) and fluid motion layouts, the application focuses heavily on user retention and reward mechanics to make the linguistic journey inherently enjoyable.

## ✨ Key Features
- **Gamified Rewards**: Uses `canvas-confetti` to celebrate user milestones and correct answers.
- **Smooth SPA Routing**: Transition between lessons, quizzes, and vocabulary banks instantly via `react-router-dom`.
- **Dynamic Micro-Interactions**: Elements scale, hover, and pop flawlessly using `framer-motion`.
- **Modern Styling System**: Built precisely utilizing the latest Tailwind CSS v4 Vite tooling.

## 🛠 Tech Stack
- Core: `React 19` & `Vite`
- UI Styling: `Tailwind CSS v4`
- Graphics/Animation: `Framer Motion`, `canvas-confetti`, `lucide-react`

## 🚀 Getting Started

```bash
# Navigate to LinguaQuest 
cd lingua-quest

# Retrieve necessary packages
npm install

# Start your language learning portal locally
npm run dev
```


## 🌐 Deployment

### Vercel / Netlify (Recommended)
1. Push this repository to GitHub.
2. Connect your GitHub account to [Vercel](https://vercel.com) or [Netlify](https://www.netlify.com).
3. Select this project and use the following settings:
   - **Build Command:** `npm run build`
   - **Output Directory:** `dist`

### GitHub Pages
1. Install the gh-pages package: `npm install gh-pages --save-dev`
2. Add deployment scripts to your `package.json`.
3. Run `npm run deploy`.
