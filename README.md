# 🗣️ Lingua Quest

**Lingua Quest** is a gamified language learning platform that bridges the gap between casual learning and professional fluency. It uses an adaptive learning engine to suggest local dialect nuances and pronunciation improvements.

### 🌟 Key Features
- **Adaptive Vocabulary**: Moves from basic greetings to complex business terminology.
- **Dialect Detection**: Specialized for localized Indian accents and regional languages.
- **Gamified Progress**: Tracking system for streaks, badges, and difficulty levels.
- **Voice Feedback**: Real-time analysis of pronunciation using Web Speech APIs.

### 🛠️ Tech Stack
- **Frontend**: React, Tailwind CSS, Framer Motion
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Voice Intelligence**: Web Speech API

### 🚀 Getting Started
```bash
npm install
npm run dev
```
